## 📋 How to Use This Repository

### Setup (5 minutes)
```bash
# You're already here!
cd servicenow-js-practice
```

### Daily Workflow (60 minutes)
1. **Open** `docs/DAY_X.md` for the lesson
2. **Read** concepts and examples (15 min)
3. **Code** practice exercises (25 min)
4. **Test** in ServiceNow instance (15 min)
5. **Log** progress in PROGRESS.md (5 min)

### Testing Your Code
```
1. Log into ServiceNow PDI
2. Navigate: System Definition > Scripts - Background
3. Copy your code into the Script field
4. Click Execute
5. View results in: System Logs - All
```

## 🆘 Help & Resources

### If You Get Stuck
1. Check `QUICK_REFERENCE.md` for syntax

3. Look at solution examples in `solutions/` folder

### Common Issues
| Problem | Solution |
|---------|----------|
| Script won't execute | Check syntax, review error in System Logs |
| Can't find System Logs | Search for "System Logs - All" table |
| GlideRecord error | Verify table exists in your instance |
| Type errors | Use `typeof` to check variable type |
