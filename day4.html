Loops - Repeat Code

Automate repetitive tasks! 🔄

📖 What is a Loop?

A loop repeats code automatically. Instead of writing the same thing 10 times, tell it "do this 10 times"!

for (var i = 1; i <= 5; i++) { gs.info('Count: ' + i); }

✨ Loop Through List

var names = ['Alice', 'Bob', 'Charlie']; for (var i = 0; i < names.length; i++) { gs.info(names[i]); }
🌍 REAL ServiceNow: Query Loop
var gr = new GlideRecord('incident'); gr.query(); while (gr.next()) { gs.info(gr.getValue('number')); }
🌟 Key Points:
  • for loop repeats a set number of times
  • while loop repeats while condition is true
  • Use break to stop the loop
  • Use continue to skip to next iteration
