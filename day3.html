If/Else - Make Decisions

Ask questions and make choices! 🧠

📖 What is If/Else?

"IF it's raining, THEN take umbrella, ELSE wear sunscreen" - that's the idea!

if (isRaining) { gs.info('Take umbrella'); } else { gs.info('No umbrella needed'); }

✨ Simple Decision

var age = 20; if (age >= 18) { gs.info('You can vote'); } else { gs.info('Too young to vote'); }
🌍 REAL ServiceNow: Check User Role
var user = gs.getUser(); if (user.hasRole('admin')) { gs.info('Admin access granted'); } else { gs.info('Limited access'); }
🌟 Key Points:
  • if (condition) { do this }
  • else { do that }
  • Use >, <, ==, >=, <= to compare
  • Use && (AND) and || (OR) for multiple conditions
