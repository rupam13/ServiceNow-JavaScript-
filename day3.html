Day 3: If/Else (Ask Questions)

Make decisions like a human! 🧠

📖 What is if/else?

Think of it like: "IF it's raining, THEN take an umbrella, ELSE wear sunscreen" You ask a question, then do something based on the answer.

var isRaining = true; if (isRaining) { gs.info('Take an umbrella'); }

✅ Practice 1: Simple Yes/No

Ask a simple question and do something:

var amHungry = true; if (amHungry) { gs.info('Eat food'); }
💡 Tip: If the question is true, do the stuff inside { }

✅ Practice 2: If/Else (Both Answers)

What if it's NOT true? Use else:

var isHot = false; if (isHot) { gs.info('Drink water'); } else { gs.info('Wear a jacket'); }

✅ Practice 3: Compare Numbers

Ask: Is 10 bigger than 5?

var age = 20; if (age >= 18) { gs.info('You can vote'); } else { gs.info('Too young to vote'); }

Symbols: > bigger, < smaller, >= bigger or equal, == equal

🎯 Challenge: Grade Checker

Check if someone passed the test:

var score = 75; if (score >= 60) { gs.info('You passed!'); } else { gs.info('You failed. Try again!'); }
🌟 Key Points:
  • if (question) { do this }
  • else { do that }
  • true = question is true
  • false = question is false
  • Symbols: >, <, >=, <=, ==, !=
