## 📚 Quick Start

### ✅ You Need:
- ServiceNow Personal Developer Instance ([Get Free PDI](https://developer.servicenow.com))
- Text editor (VS Code recommended)
- 1 hour per day for 10 days
- This repository

### 🚀 Start Now:
1. Open `10_DAY_CHALLENGE.md` for the complete schedule
2. Read `00_setup/GETTING_STARTED.md` for setup instructions
3. Start with `01_basics/day_01_getting_started.js`
4. Follow the daily lessons sequentially

### 📋 Track Progress:
- Update `PROGRESS.md` daily
- Use `QUICK_REFERENCE.md` for syntax lookups
- Test every code snippet in ServiceNow

## 🆘 Troubleshooting

| Issue | Solution |
|-------|----------|
| Script won't execute | Check System Logs - All for errors |
| Can't find System Logs | Search for "System Logs" in search bar |
| GlideRecord not found | Verify table name exists (case-sensitive) |
| Type errors | Use `typeof` to check variable type |
| Out of time? | Focus on core concepts, extend later |
