GlideRecord - Query Database

Access ServiceNow data! 🗄️

📖 GlideRecord Basics

GlideRecord queries tables and retrieves data from ServiceNow.

var gr = new GlideRecord('incident'); gr.query(); while (gr.next()) { gs.info(gr.getValue('number')); }
🌍 REAL ServiceNow: High Priority Incidents
var gr = new GlideRecord('incident'); gr.addQuery('priority', '1'); gr.query(); gs.info('Critical incidents: ' + gr.getRowCount());
🌟 Key Points:
  • new GlideRecord(table)
  • addQuery() filters results
  • query() executes
  • getValue() gets field value
