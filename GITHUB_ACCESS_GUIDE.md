## 💻 How to Copy Code from GitHub

### Option 1: Copy-Paste
```
1. Open the file on GitHub
2. Select the code you want
3. Press Ctrl+C to copy
4. Go to ServiceNow
5. Paste into Script field
```

### Option 2: Download Entire Repo
```
1. Click green "Code" button
2. Select "Download ZIP"
3. Extract to your computer
4. Open files in your text editor
```

### Option 3: Clone to Your Computer
```bash
# Open terminal/Git Bash
git clone https://github.com/rupam13/ServiceNow-JavaScript-.git

# Navigate to folder
cd ServiceNow-JavaScript-

# Now all files are on your computer
# Open with VS Code or any editor
```
