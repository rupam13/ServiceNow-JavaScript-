📖 What is an Object?

Objects store related information with labels.

var person = { name: 'Alice', age: 25 }; gs.info(person.name); gs.info(person.age);
🌟 Key Points:
  • var obj = {key: value}
  • Access with obj.key
  • Multiple properties
  • Can store any type
