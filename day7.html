Day 7: Objects

📖 What is an Object?

An object stores related information with labels.

var user = { name: 'Alice', age: 25, email: 'alice@example.com' }; gs.info(user.name); // Alice gs.info(user.email); // alice@example.com
🌍 REAL ServiceNow: User Object
var currentUser = gs.getUser(); var userInfo = { name: currentUser.getName(), id: currentUser.getID(), email: currentUser.getEmail(), role: 'admin' }; gs.info('User: ' + userInfo.name); gs.info('Email: ' + userInfo.email);

✅ Practice: Incident Object

var incident = { number: 'INC0001234', state: 'new', priority: 2, assigned_to: 'John' }; gs.info(incident.number); gs.info('State: ' + incident.state);
🌍 Check Role-Based Restrictions:
var userRoles = { canEdit: false, canDelete: false, canView: true, canCreate: false }; if (userRoles.canEdit) { gs.info('User can edit'); } else { gs.info('User cannot edit (View only)'); }
