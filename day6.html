Arrays - Store Lists

Manage multiple items! 📋

📖 What is an Array?

Arrays store multiple items in one variable like a shopping list.

var fruits = ['Apple', 'Banana', 'Orange']; gs.info(fruits[0]); // Apple gs.info(fruits.length); // 3
🌟 Key Points:
  • var arr = [item1, item2]
  • Access with arr[0], arr[1]...
  • push() adds to end
  • length shows size
