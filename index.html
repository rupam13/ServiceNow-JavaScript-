📖 What is gs.info()?

Think of gs.info() as a megaphone for your code. You tell the computer something, and it prints it out for you to see!

gs.info('Hello World');

This will print: Hello World

✨ Your First Practice

Type this simple code in the editor on the right:

gs.info('Hello');

👉 Then click the ▶ RUN button to see the magic happen!

💡 Pro Tip: Everything you want to print must be inside quotes like 'this' or "this"

🎯 Practice 2: Say Your Name

Try changing the code to say your own name:

gs.info('My name is Alice');

🎯 Practice 3: Multiple Lines

You can print many things, one after another:

gs.info('Hello'); gs.info('My name is Bob'); gs.info('I am learning ServiceNow');
🌟 Key Takeaways:
  • gs.info() prints text to the output
  • Text must be in quotes
  • Each line prints separately
  • You can print as many times as you want
