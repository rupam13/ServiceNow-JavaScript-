Day 1: Writing Your First Code

Learn to say "Hello" to ServiceNow 👋

📖 What is gs.info()?

Think of gs.info() as a megaphone. You tell the computer something, and it prints it out so everyone can see!

gs.info('Hello World');

✅ Practice 1: Say Hello

Type this in the code editor on the right:

gs.info('Hello');

👉 Click "Run" button to see what happens!

💡 Tip: Everything you want to print must be in quotes: 'like this'

✅ Practice 2: Say Your Name

Try this:

gs.info('My name is Alice');

✅ Practice 3: Say Multiple Things

You can say many things one after another:

gs.info('Hello'); gs.info('My name is Bob'); gs.info('I like coding');

🎯 Challenge: About You

Write 3 lines about yourself! For example:

gs.info('Hello'); gs.info('My name is Sarah'); gs.info('I love learning');

Change the text to match YOU! Then click Run.

🌟 Key Points:
  • gs.info() prints text
  • Text must be in quotes
  • You can use many gs.info() lines
  • Each line prints separately
