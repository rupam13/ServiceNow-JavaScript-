Day 2: Variables (Save Things in Boxes)

Think like a piggy bank 🏦

📖 What is a Variable?

A variable is like a box with a label. You put something inside, close the lid, and use the label to remember what's inside!

var name = 'Alice'; gs.info(name);

✅ Practice 1: Make Your First Box

Create a box with your name:

var myName = 'Bob'; gs.info(myName);
💡 Tip: var = make a box, name = label, 'Alice' = what's inside

✅ Practice 2: Combine Two Boxes

Mix two boxes together:

var firstName = 'John'; var lastName = 'Doe'; var fullName = firstName + ' ' + lastName; gs.info(fullName);

✅ Practice 3: Math with Boxes

You can do math with numbers in boxes:

var apple = 5; var orange = 3; var total = apple + orange; gs.info(total);

🎯 Challenge: Your Profile

Create a profile about yourself:

var name = 'Your Name'; var age = 25; var city = 'Your City'; gs.info('Name: ' + name); gs.info('Age: ' + age); gs.info('City: ' + city);
🌟 Key Points:
  • var = make a box
  • label = what you call it (name, age, city)
  • value = what's inside ('Alice', 25, etc.)
  • + combines things
  • You can use the box many times
