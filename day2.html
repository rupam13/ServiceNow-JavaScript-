Variables - Save Data in Boxes

Think like a piggy bank 🏦

📖 What is a Variable?

A variable is like a labeled box. You put something inside, close the lid, and use the label to remember what's inside!

var name = 'Alice'; gs.info(name); // Prints: Alice

✨ Create Your First Variable

Try this in the editor:

var myName = 'Bob'; gs.info(myName);
💡 Pro Tip: var creates a box, the name is your label, and the value is what's inside!

🎯 Combine Variables

Mix variables together with the + operator:

var firstName = 'John'; var lastName = 'Doe'; var fullName = firstName + ' ' + lastName; gs.info(fullName);
🌍 REAL ServiceNow: Get User Info
var user = gs.getUser(); var userName = user.getName(); var userEmail = user.getEmail(); gs.info('User: ' + userName);
🌟 Key Points:
  • Variables store information
  • Use var to create a variable
  • Use + to combine text
  • You can reuse variables
