📖 What is a Function?

A function is a reusable block of code that does ONE job.

function greet(name) { gs.info('Hello, ' + name); } greet('Alice'); greet('Bob');
🌍 REAL ServiceNow Use Case:

Check if user has specific role:

function userHasRole(roleName) { var gr = new GlideRecord('sys_user_role'); gr.addQuery('user', gs.getUser().getID()); gr.addQuery('role.name', roleName); gr.query(); return gr.getRowCount() > 0; } if (userHasRole('admin')) { gs.info('User is admin'); }
